# Plugin Integration
mcMMO has limited support for some optional features when used with other plugins, such as World Guard.

> Feel free to ask for support for integration with other plugins in the discord, or to expand existing support, I'm always open to suggestions

# World Guard

mcMMO registers three custom WorldGuard `StateFlag` values for per-region control of skills, XP gain, and hardcore/vampirism penalties.

See the [WorldGuard Integration](/plugin-integration/worldguard) page for the full flag reference and usage instructions.

## PAPI / Placeholders

The latest versions of mcMMO have built in support for PAPI, no need to download anything from the PAPI ecloud.

See the [PlaceholderAPI Placeholders](/placeholders) page for the full list of supported placeholders, including the skill, party, XP event, and leaderboard placeholders.
